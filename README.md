# 📱 Social Media Addiction Analysis Dashboard

## 📌 Project Overview
This project analyzes **social media usage behavior** and its impact on **sleep patterns, relationships, and academic performance**.  
The dashboard provides meaningful insights into how excessive usage can lead to potential addiction and lifestyle imbalance.

---

## 🎯 Objectives
- Analyze daily social media usage trends  
- Study the relationship between sleep and screen time  
- Identify most used platforms  
- Understand the impact on academic performance  
- Explore demographic patterns (age, gender, country)

---

## 📂 Dataset Information
The dataset includes:
- Age  
- Gender  
- Social Media Platform  
- Daily Usage Hours  
- Sleep Hours  
- Relationship Status  
- Academic Performance  
- Country  

---

## 📊 Key KPIs
- **Average Sleep:** 6.87 hours  
- **Average Usage:** 4.92 hours/day  
- **Total Users Analyzed:** 705  

---

## 📈 Dashboard Insights

### 📊 Average Daily Usage
- Usage ranges between **3.8 to 5.7 hours/day**  
- Indicates moderate to high social media engagement  

### 😴 Sleep vs Usage
- Higher usage is associated with **lower sleep hours**  
- Clear negative correlation between screen time and sleep  

### ❤️ Relationship Status
- Majority users are **Single**  
- Followed by users in relationships  

### 📱 Most Used Platforms
- **TikTok** is the most used platform  
- Followed by WhatsApp and Instagram  
- Professional platforms like LinkedIn are least used  

### 🎂 Age Distribution
- Majority users fall between **19–22 age group**  
- Young adults are most active on social media  

### 🌍 Top Countries
- Highest users from:
  - India  
  - USA  
  - Canada  
- Shows global usage trend  

### 🎓 Academic Impact
- Significant number of users report:
  - Negative impact on academic performance  
- Indicates addiction affecting productivity
